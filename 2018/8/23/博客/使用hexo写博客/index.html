使用hexo写博客

发表于 2018-08-23 | 分类于 博客 | 阅读次数:

　　距离上次搭建博客过去了8个月，现在想重新把博客捡起来，做一些日常的记录，总结。由于之前博客已经搭建完成了，所以这里只是单纯的做一些总结回顾，记录下搭建的大概的流程。
　　

博客搭建

　　博客的搭建过程主要参考官方文档

  1. 环境配置
    • 安装Git
    • 安装Node.js
    • 安装Hexo
  2. 建站、网站基本配置
    1. 在指定的文件夹中建站
      hexo init
      nmp install
    2. 网站基础配置
      修改root目录下的_config.yml文件，详见
    3. 文件夹结构
      • _config.yml : 网站配置信息
      • package.json: 应用程序信息
      • scaffolds: 模板文件夹
      • source: 资源文件夹
      • theme: 主题文件夹

主题配置

　　主题的配置参考next官方文档

  1. 从git上下载next
  2. 在站点设置文件中配置主题
  3. 修改next的配置文件
  4. 集成常用三方服务
    1. 百度统计
    2. LeanCloud 页面统计

内容书写

　　以上配置完成之后，就可以运行看一下配置的效果。没有什么异常的情况就可以参考操作命令和MarkDown语法进行基本的内容的书写了。

  1. 生成新文章
    hexo new [title]

  2. 编辑

  3. 生成静态页面

    hexo generate 或 hexo g

部署

　　在本地确认没有问题之后，就可以部署到git上，首先在站点配置文件_config.yml中进行git的相关配置，然后再通过命令 hexo d 进行部署即可。
